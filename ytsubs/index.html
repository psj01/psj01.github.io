How to Get Your YouTube Subscriptions File

  1. Go to Google Takeout and make sure you’re logged into the correct Google/YouTube account.
  2. Under Create a new export, choose YouTube and YouTube Music.
  3. Click All YouTube data included and select only Subscriptions.
  4. Click Next step, choose Export once, then click Create export.
  5. Wait for the export to finish, then download the archive.
  6. Extract the archive and find the file named subscriptions.csv.
  7. Upload that file below to see your subscriptions.
